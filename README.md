## About

IRO (Issues Reporting Outlet) is a web-based solution for relating and reporting issues
concerning AAUA students and indigenes of the host communities — Akungba, Iwaro, Etioro,
and Ayegunle.

## Contents of this repository

| File | Description |
|---|---|
| `index.html` | Mini web interface — report an issue, view the public issue feed |
| `ishikawa_diagram.png` | Ishikawa (fishbone) root-cause analysis of the reporting problem |
| `usecase_diagram.png` | Use-case diagram showing system actors and interactions |
| `SRS_IRO.docx` | Software Requirements Specification — functional & non-functional requirements |
